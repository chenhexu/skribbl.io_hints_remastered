# Privacy Policy — Skribbl Auto Guesser

**Last updated:** 2025

This extension **does not collect any user data**. The Chrome Web Store requires a privacy policy when an extension collects user data; because this extension does not collect data, a policy is not strictly required. This page is provided for transparency and to satisfy the store’s optional privacy policy URL field.

## Data collection

**None.** Skribbl Auto Guesser does not collect, transmit, or sell any personal or usage data.

## What is stored locally

The extension stores the following **only on your device** (in your browser’s local storage), so it can work across sessions:

- Your preferences (e.g. panel position, delay, dot prefix, refresh rates)
- Optional custom word list and cache

This data never leaves your device and is not sent to any server or third party.

## Permissions

- **Host access (skribbl.io):** Used only to read the game hint and guess input on skribbl.io so the extension can suggest words and send guesses. It does not run on other sites.
- **Storage:** Used only to save the above preferences and cache locally. No data is transmitted.

## Changes

If this policy changes, the “Last updated” date will be revised.

## Contact

For questions, open an issue on the project’s GitHub repository or use the contact email on the Chrome Web Store listing.
