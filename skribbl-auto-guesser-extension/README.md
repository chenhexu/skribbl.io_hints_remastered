# Extension moved to standalone project

The Chrome extension lives in a **separate project** so it can be developed (and e.g. extended with auto-drawer) independently:

- **Path:** `c:\Projects\skribbl-auto-guesser-extension` (sibling folder to `Skribbl_hints`)
- **Or:** Add that folder as a second root in your workspace.

All files have been copied there. Use that folder for loading unpacked, building the ZIP, and future work. The userscript source remains here: `skribbl-hints-app/public/userscripts/skribbl-auto-guesser.user.js`.
